# Laundry Services Hero Section

This project is a simple web page that demonstrates the creation of a Hero Section for a laundry service website.

## Features
- Navigation bar with menu items
- Logo on left and user name on right
- Hero section divided into two parts:
  - Left: Heading, description and button
  - Right: Image
- Fully responsive layout using viewport units
- No scrolling page

## Technologies Used
- HTML5
- CSS3

## Project Structure

LaundryTask
│
├── index.html
├── style.css
└── README.md

## Setup Instructions

1. Download or clone the project folder.
2. Make sure all files are in the same directory:
   - index.html
   - style.css
3. Open `index.html` in any web browser.
4. The webpage will load with navbar and hero section.

## Important Notes
- Flexbox is not used
- Inline / inline-block layout is used
- Viewport units (vh, vw) are applied
- Page does not scroll
